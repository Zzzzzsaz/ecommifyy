# Ecommify - Panel E-Commerce Dashboard

## Wersja 2.1 (Feb 23, 2026)

### UI/UX
- **Mobile**: Bottom nav (Start, Wyniki, Zadania, Pomysły, Więcej)
- **Desktop**: Fixed sidebar 240px
- **Light theme** - profesjonalny biały design
- **Responsive** - tabele na desktop, karty na mobile

### Dashboard
- Date picker jak Shopify (Dzisiaj, Wczoraj, Ten tydzień, Ten miesiąc, Poprzedni miesiąc)
- Karty: Przychód, Koszty reklam, Zysk, Na osobę
- Cel miesięczny z progress bar
- Przypomnienia

### Wyniki
- Kolumny: Dzień, Przychód, Koszty, **Zysk**, **Na osobę**
- Mobile: karty responsywne
- Przyciski (+) do dodawania przy każdym dniu

### Zadania
- Zakładki: Aktywne, Ukończone, **Usunięte**
- Soft delete z możliwością restore
- Priorytet, przypisanie, termin

### Pomysły
- Lista z kategoriami
- Gwiazdka ważności
- Filtry

## Technical Stack
- React + Tailwind CSS + Shadcn/UI
- FastAPI + MongoDB Atlas
- Railway.app deployment
